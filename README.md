# oyun-kutuphanesi-uygulamasi 🎮

Bu proje, video oyun koleksiyonlarını dijital ortamda düzenli bir şekilde takip etmek, yönetmek ve arşivlemek amacıyla geliştirilmiş profesyonel bir masaüstü kataloğu uygulamasıdır.

## 📝 Proje Hakkında
**oyun-kutuphanesi-uygulamasi**, kullanıcıların sahip oldukları oyunları tür, çıkış yılı, geliştirici ve platform gibi detaylı kriterlere göre kategorize etmelerini sağlar. Şık bir kullanıcı arayüzü ile modern bir oyun kütüphanesi deneyimi sunan uygulama, veri yönetimini JSON tabanlı bir mimariyle gerçekleştirerek taşınabilirlik ve erişilebilirlik sunar.

## ✨ Öne Çıkan Özellikler
- 🔍 **Gelişmiş Arama:** Oyunları başlık, tür veya etiketlere göre anlık olarak filtreleme.
- 🗂️ **Kategorizasyon:** Oyunları türlerine göre otomatik gruplandırma ve görsel kartlar halinde sunma.
- ⚙️ **Detaylı Yönetim:** Oyun ekleme, düzenleme ve silme işlemlerini içeren tam CRUD desteği.
- 📂 **Veri Aktarımı:** Koleksiyonu JSON formatında içeri aktarma (Import) ve dışarı aktarma (Export) özelliği.
- 🖼️ **Görsel Katalog:** Oyun kapak görselleriyle zenginleştirilmiş modern ve dinamik arayüz.

## 🛠️ Kullanılan Teknolojiler ve Kütüphaneler
- **Programlama Dili:** Java 21
- **Arayüz Framework:** JavaFX (Modern ve interaktif UI bileşenleri için)
- **Veri Yönetimi:** Google Gson (JSON serileştirme ve deserileştirme işlemleri için)
- **Derleme Aracı:** Gradle (Bağımlılık yönetimi ve otomasyon için)

## 🤖 Geliştirme Süreci
Bu projenin mimarisi ve geliştirme aşamaları, **insan-makine iş birliği** (Human-AI Collaboration) prensibiyle bir **Yapay Zeka Ajanı** desteğiyle gerçekleştirilmiştir. Tasarım kararları, kod optimizasyonu ve hata ayıklama süreçlerinde üst düzey yapay zeka algoritmalarından yararlanılarak, endüstri standartlarına uygun, temiz ve sürdürülebilir bir kod yapısı (Clean Code) hedeflenmiştir. Bu yaklaşım, modern yazılım geliştirme metodolojilerinin en güncel örneklerinden birini temsil etmektedir.

## 🚀 Kurulum ve Çalıştırma Talimatları
Uygulamayı yerel ortamınızda çalıştırmak için şu adımları izleyin:

1. **Önkoşullar:** Bilgisayarınızda **JDK 21** yüklü olmalıdır.
2. **Depoyu Klonlayın:**
   ```bash
   git clone https://github.com/kullanici-adi/oyun-kutuphanesi-uygulamasi.git
   cd oyun-kutuphanesi-uygulamasi
   ```
3. **Uygulamayı Çalıştırın:** Gradle wrapper kullanarak projeyi başlatın:

# oyun-kutuphanesi-uygulamasi 🎮

This project is a professional desktop catalog application developed to organize, manage, and archive video game collections in a digital environment.

## 📝 About the Project
**oyun-kutuphanesi-uygulamasi** allows users to categorize their games based on detailed criteria such as genre, release year, developer, and platform. Offering a modern game library experience with a sleek user interface, the application provides portability and accessibility by performing data management with a JSON-based architecture.

## ✨ Key Features
- 🔍 **Advanced Search:** Instant filtering of games by title, genre, or tags.
- 🗂️ **Categorization:** Automatic grouping of games by genre and presentation in visual cards.
- ⚙️ **Comprehensive Management:** Full CRUD support including adding, editing, and deleting games.
- 📂 **Data Transfer:** Ability to Import and Export the collection in JSON format.
- 🖼️ **Visual Catalog:** A modern and dynamic interface enriched with game cover images.

## 🛠️ Technologies Used and Libraries
- **Programming Language:** Java 21
- **Interface Framework:** JavaFX (For modern and interactive UI components)
- **Data Management:** Google Gson (For JSON serialization and deserialization)
- **Build Tool:** Gradle (For dependency management and automation)

## 🤖 Development Process
The architecture and development phases of this project were carried out using **Human-AI Collaboration** principles, supported by an **AI Agent**. By utilizing high-level AI algorithms in design decisions, code optimization, and debugging processes, we aimed for a clean and sustainable code structure in accordance with industry standards. This approach represents one of the most up-to-date examples of modern software development methodologies.

## 🚀 Installation and Running Instructions
Follow these steps to run the application in your local environment:

1. **Prerequisites:** **JDK 21** must be installed on your computer.
2. **Clone the Repository:**
   ```bash
   git clone https://github.com/username/oyun-kutuphanesi-uygulamasi.git
   cd oyun-kutuphanesi-uygulamasi
   ```
3. **Run the Application:** Start the project using the Gradle wrapper:
   ```bash
   ./gradlew run
   ```
